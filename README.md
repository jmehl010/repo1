# Programming Practice

Using the greatest programming language ever, python.

## Install

Create a venv:
```
python -m venv .\.venv
```

Activate the venv:
```
.\.venv\Scripts\activate
```

install the required python libs:
```
pip install requirements.txt
```

## Usage

```
python main.py
```
